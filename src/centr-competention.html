Центр экспертизы Sabre оказывает услуги поддержки, бизнес-консалтинга и технической аналитики по программным продуктам производственных платформ Sabre AirCentre и AirVision.

  • Планирование сезонного расписания
  • Оперативное управление расписанием
  • Формирование планов полетов
  • Управление экипажами
  • Управление центровкой и контроль загрузки
  • Sabre Schedule Manager
    Управление сезонным расписанием полетов
  • Sabre Movement Manager
    Контроль расписания и выполнения полетов в режиме реального времени
  • Sabre Recovery Manager Ops
    Оптимизация и поддержка принятия решений при управлении производством полетов
  • Sabre Flight Plan Manager
    Формирование плана полетов с учетом необходимых условий и ограничений
  • Sabre Crew Manager
    Планирование летных и кабинных экипажей
  • SabreSonic CSS Load Manager
    Управление загрузкой воздушных судов
  • Опыт внедрения масштабных IT-решений в ведущих авиационных предприятиях России;
  • Оптимальный подбор программных продуктов для решения задач заказчиков;
  • Выделенная локальная команда сертифицированных русскоговорящих отраслевых экспертов;
  • Прямая поддержка вендора, гарантирующая минимизацию риска срыва сроков и оптимизацию расходов на проект.
  • Политика обработки персональных данных
  • Политика обработки персональных данных по поручению
  • Изменения политики лицензирования ПО производства АО "Интегро Текнолоджиз"
